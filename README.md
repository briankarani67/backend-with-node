I am learning how to build servers, APIs, and connect applications to databases while understanding how real backend systems work.

* Node.js
* Express.js
* Postman for API testing

This project currently includes a basic Express server that responds to HTTP requests.

## How to Run the Project

1. Clone the repository

git clone https://github.com/your-username/backend-journey.git

2. Navigate into the project folder

cd backend-journey

3. Install dependencies

npm install

4. Start the server

nodeserver.js

The server will run on:

http://localhost:2000
